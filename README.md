## AndroidScript

安卓自动化脚本，辅助开发者处理国际化问题。

#### 功能
* 支持代码中（Java、Kotlin、xml）扫描并自动生成（添加）到strings.xml中。可扩展为支持iOS文案扫描。
* 支持string.xml转换为Excel文件
* 支持Excel转换为各个语言string.xml
* 支持全局key替换
* 支持通用字符串替换，高度定制化
* 支持扫描并删除无用Drawable
* 支持扫描并删除无用strings.xml的语言配置
